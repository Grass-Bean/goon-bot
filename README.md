- **Utility Commands**

- **Fun Commands**
  - Loans
  - Slot machine

Many other commands.

1. **Clone the repository**
   ```
   git clone https://github.com/Grass-Bean/goon-bot.git
   cd GoonBot
   ```
2. **Install dependencies**
    ```
    npm install
    ```

4. **Start the bot**
    ```
    node index.js
    ```
